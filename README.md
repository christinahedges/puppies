# puppies

Crucial repo for all projects.

## Installation

```
pip install puppies
```

## Usage

```python
import puppies
pup = puppies.Puppy(breed='corgi')
pup.sit()     # Use sit method on Puppy
pup.stay()     # Use stay method on Puppy
```
